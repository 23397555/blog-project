# MyBlog - A React Web Application

## Student Details

- **Student Name:** Mihir
- **Project Title:** MyBlog - A React Web Application

---

# Objective

The objective of this project is to develop a responsive blog web application using **ReactJS**, **Tailwind CSS**, and **Vite**. The project focuses on understanding the fundamentals of modern frontend development, including component-based architecture, state management, API integration, responsive UI design, and local data storage.

---

# Project Features

## 1. Component-Based Architecture
The application is developed using reusable React components to maintain clean code and improve scalability.

Components include:
- Navbar
- Blog
- Categories
- Subscription Form
- Footer

---

## 2. Responsive User Interface
- Built using **Tailwind CSS**
- Mobile-friendly responsive layout
- Glassmorphism design
- Modern card-based blog layout

---

## 3. State Management
React Hooks are used for efficient state handling.

- `useState`
  - Manage component state
  - Handle form inputs
  - Store blog data

- `useEffect`
  - Fetch API data
  - Load and save Local Storage data
  - Execute lifecycle operations

---

## 4. Data Handling
- Fetch blog data from APIs
- Store user preferences using Local Storage
- Render dynamic content efficiently

---

# Lab Syllabus Coverage

The project covers the following ReactJS laboratory topics:

| Lab Topic | Implementation |
|-----------|----------------|
| Component Architecture | Organized project into reusable components such as Navbar, Blog, Categories, Footer, and Form |
| State & Props | Passed data dynamically between parent and child components |
| React Hooks | Used `useState` and `useEffect` for state management and lifecycle methods |
| Tailwind CSS | Designed responsive layouts, cards, headers, footers, and sections |
| Controlled Forms | Implemented subscription form with input validation and state handling |
| Data Tables | Created structured tables for displaying information |
| API Integration | Fetched and displayed external blog data |
| Lists & Keys | Rendered categories and blog posts dynamically using lists |
| Local Storage | Stored and retrieved user data for persistence |
